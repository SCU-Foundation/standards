# Lattice Web (LW) — Changelog

## v1.0 — Initial Draft Standard
- First publication of the Lattice Web standard  
- Defines node classes and node‑level metadata  
- Defines edge classes and relational metadata  
- Defines graph envelopes and graph‑level metadata  
- Defines graph‑level validity rules  
- Defines graph‑level boundaries and cross‑domain coherence  
- Establishes LW as the graph‑level semantic substrate of the Lattice ecosystem  
- Aligned with TS v1.0, LP v1.0, and LM v1.0  
- Added canonical metadata and publication structure
