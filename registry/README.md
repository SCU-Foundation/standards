# SCU Standards Registry  
This folder contains the canonical registry of all standards governed by the SCU Foundation.

Contents:
- `registry.md` — human-readable registry  
- `registry.json` — machine-readable registry  
- `metadata.json` — field definitions and registry metadata  
- `checksums.txt` — integrity verification hashes  
- `STATUS.md` — lifecycle and maturity  
- `CHANGELOG.md` — version history  
- `VERSION` — current registry version  
- `LICENSE` — open license  

The Standards Registry is maintained by the SCU Foundation and updated through the formal governance process.
