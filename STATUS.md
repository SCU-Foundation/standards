# Standards Repository — Status

- **Repository:** Standards  
- **Scope:** Normative standards of the SCU ecosystem  
- **Era:** Ecosystem Era (vE1.x)  
- **Maintained by:** SCU Foundation  

## Role in the Ecosystem

This repository contains the **final, normative standards** that define:

- semantic substrate (LP)  
- metadata layer (LW)  
- meta‑semantic layer (LM)  
- identity (LIT)  
- safety & certification (LCSL)  
- trust, cognition, and governance (TS)  

These standards are **normative**, **versioned**, and **stand‑alone**.  
They MUST NOT depend on GitHub structure or canonical document layout.
