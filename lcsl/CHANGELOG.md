# LCSL v1.0 — Changelog

## v1.0 — Initial Publication
- First publication of the Lattice Certification & Safety Layer (LCSL) Standard.
- Establishes normative rules for certification, safety, attestation, provenance, domain transitions, capability boundaries, version floors, update sequencing, and compliance.
- Defines conformance requirements for all Lattice‑conformant devices, extensions, runtimes, capabilities, and agents.
