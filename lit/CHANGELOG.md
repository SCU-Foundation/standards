# Lattice Identity Token (LIT) — Changelog

## v1.0 — Initial Draft Standard
- first publication of the Lattice Identity Token standard  
- defines identity primitives  
- defines identity metadata  
- defines identity binding rules  
- defines identity boundaries  
- defines identity attestation rules  
- defines identity revocation rules  
- establishes LIT as the identity substrate of the Lattice ecosystem  
- aligned with TS v1.0, LP v1.0, LM v1.0, and LW v1.0  
- added canonical metadata and publication structure
